# Kami's Gamified English Resources

A multi-page English learning web application with gamified Kahoot and Wordwall resources.

## Project Goal

The purpose of this project is to provide engaging digital English learning resources for students and teachers.

## Planned Features

- User registration and login
- Resource catalogue
- Resource details pages
- Kahoot and Wordwall learning activities
- Admin panel
- File upload and download
- Responsive design for desktop and mobile devices

- Bootstrap
